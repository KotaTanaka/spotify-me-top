# spotify-me-top

## Tech Stack

- Language ... TypeScript
- Framework ... [Remix](https://remix.run/docs)
- External API ... [Spotify Web API](https://developer.spotify.com/documentation/web-api)

## Requirements

- Node.js `v16.x`

## Getting Started

```sh
# Install packages
yarn

# Serve at http://localhost:3000
yarn dev
```
